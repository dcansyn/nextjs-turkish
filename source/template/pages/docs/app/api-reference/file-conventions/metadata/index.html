Metadata Files API Reference

Dokümanların bu bölümü Metadata dosya kurallarını kapsamaktadır. Dosya tabanlı meta veriler, rota segmentlerine özel meta veri dosyaları eklenerek tanımlanabilir.

Her dosya kuralı statik bir dosya (örn. opengraph-image.jpg) veya dosyayı oluşturmak için kod kullanan dinamik bir varyant (örn. opengraph-image.js) kullanılarak tanımlanabilir.

Bir dosya tanımlandıktan sonra Next.js dosyayı otomatik olarak sunacak (önbelleğe alma için üretimde hash'lerle) ve ilgili başlık öğelerini varlığın URL'si, dosya türü ve görüntü boyutu gibi doğru meta verilerle güncelleyecektir.
